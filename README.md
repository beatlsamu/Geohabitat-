# Geohabitat-
Dashboard web 3D que analiza habitabilidad global por país. Cruza clima (Open-Meteo, NASA POWER), calidad del aire (OpenAQ), población y PIB (World Bank), y energía renovable (CNE Chile) para calcular scores explicables de habitabilidad, riesgo, presión humana y vulnerabilidad futura. Globo interactivo con D3, estética atlas científico vintage.
